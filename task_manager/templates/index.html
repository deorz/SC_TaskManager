ИС «Диспетчер суперкомпьютера»

Задачи

Вы можете создавать задачу, указывать количество ядер(потоков) выполнение, а система всё сделает за вас.

Приорет задач

Приоритет задачи и количество процессорных ядер используются для автоматического управления выполнением задач. Т.е. если есть свободные процессорные ядра из очереди берется самая приоритетная задача.

Стэк

Сервис разработан при помощи Django Rest Framework и использует базу данных MySQL.

Для фронтенда используется набор классов Bootstrap и Bootstrap JavaScript.

Вызовы API реализованы при помощи JavaScript fetch()
